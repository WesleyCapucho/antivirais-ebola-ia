# Extended Data and Code Availability Guidelines

Para revistas de alto impacto, a reprodutibilidade aberta é obrigatória.

## 1. Code Availability
Todo o código customizado (incluindo o pipeline de extração de dados, arquitetura TensorFlow `CW-MSE` e scripts PyMOL) foi depositado no GitHub.
- **Link do Repositório:** `https://github.com/WesleyCapucho/antivirais-ebola-ia`
- **Ambiente de Treinamento:** O Jupyter Notebook utilizado para treinamento da IA encontra-se no repositório (`Colab_Ebola_DeepLearning_Pipeline.ipynb`).

## 2. Data Availability
Os datasets em formato bruto (raw) e curado (Tier 1) estão disponíveis na sub-pasta `data/` do repositório GitHub.
- As matrizes completas de Risco de Viés (Risk of Bias) estão fornecidas como "Supplementary Table 1".
- Arquivos de conformação 3D (`.sdf`) e proteínas alvo (`.pdb`) estão disponíveis mediante solicitação ou acessíveis via PDB Accession IDs listados nos Métodos.

## 3. Protocol Registry
A revisão sistemática inicial seguiu as diretrizes PRISMA e o protocolo original foi submetido no formato padrão (PROSPERO).
